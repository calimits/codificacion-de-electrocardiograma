## üß© Problem
Electrocardiograms are analog signals that must be digitized for storage, analysis, and transmission.  
The challenge was to understand how **different levels of quantization (bit depth)** impact the fidelity of the ECG signal.

---

## üí° Solution
This project implements an interactive web application that:
- Encodes ECG signals using varying bit depths.  
- Decodes and reconstructs the signal for visualization.  
- Compares the quality of the reconstructed signal against the original.  
- Provides a clear demonstration of the trade-off between **precision** and **storage efficiency**.
